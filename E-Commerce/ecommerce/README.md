# 🧾 Angular Expense Tracker

An **Expense Tracker Application** built with Angular, Chart.js, and a mock backend (MockAPI/json-server).  
This project helps users record, categorize, and visualize expenses with CRUD operations and charts.

---

## 🚀 Features
- **Add Expense** – Form to input description, category, amount, and date.
- **Edit Expense** – Update existing expense details with pre-filled form values.
- **Delete Expense** – Remove unwanted transactions.
- **Expense Summary** – Calculate and display total expenses dynamically.
- **Charts Dashboard** – Visualize spending trends using Pie/Bar charts.
- **Mock Backend** – Integrated with `json-server` or [MockAPI.io](https://mockapi.io) for REST API simulation.
- **Responsive UI** – Clean layout with Angular Material/Bootstrap.

---

## 🛠️ Tech Stack
- **Angular CLI** – Project setup and scaffolding
- **Chart.js / ng2-charts** – Data visualization
- **Reactive Forms** – Input handling and validation
- **MockAPI / json-server** – Backend simulation
